# User Guides

## Super Admin

1. Log in with Super Admin credentials.
2. Review dashboard cards for users, activity, system health, security, and performance.
3. Manage admins, users, curriculum, reports, files, notifications, settings, security, backups, and restores.
4. Use Audit Logs and Security Center for accountability.
5. Use Profile and Settings to maintain account details.
6. Log out after administrative work.

## Admin

1. Log in from the Admin portal.
2. Use the dashboard to monitor teachers, students, curriculum, materials, reports, and announcements.
3. Create and maintain teachers and students.
4. Manage curriculum and learning material workflows.
5. Review reports and export data when authorized.
6. Manage notifications and announcements.
7. Log out after work.

## Teacher

1. Log in from the Teacher portal.
2. Review teaching dashboard, students, curriculum progress, and analytics.
3. Upload materials, notes, videos, and supporting content.
4. Mark topics completed when lessons are delivered.
5. Generate AI questions for taught topics.
6. Review quiz analytics, student results, and question exposure.
7. Manage profile and notifications.
8. Log out after teaching tasks.

## Student

1. Log in from the Student portal.
2. View dashboard, materials, notes, videos, announcements, and notifications.
3. Start quizzes only for eligible completed topics.
4. Save answers and submit quiz attempts.
5. Review results, marks, reports, and quiz history.
6. Upload assignment videos when required.
7. Manage profile and settings.
8. Log out after use.
